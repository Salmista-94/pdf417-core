enum PPDetectionStatus

Defines a status of the object detection process.

Enumerator
DETECTION_STATUS_SUCCESS 

Object has been detected successfully

DETECTION_STATUS_FAIL 

Detection failed, nothing detected

DETECTION_STATUS_CAMERA_TOO_HIGH 

Object detected, but the camera is too far above it

DETECTION_STATUS_QR_SUCCESS 

QR code detected

DETECTION_STATUS_PARTIAL_OBJECT 

Object has been detected, but some parts of it are not in image

DETECTION_STATUS_CAMERA_AT_ANGLE 

Object has been detected, but camera is at too big angle

DETECTION_STATUS_CAMERA_TOO_NEAR 

Object detected, but the camera is too close to the object

enum RawImageType

Enumeration of all possible raw image types.

Enumerator
RAW_IMAGE_TYPE_BGRA 

4 channel raw image with 32 bit per pixel, 8 bit per channel, channel order is BGRA

RAW_IMAGE_TYPE_BGR 

2 channel raw image with 24 bit per pixel, 8 bit per channel, channel order is BGR

RAW_IMAGE_TYPE_GRAY 

1 channel grayscale raw image with 8 bit per pixel

RAW_IMAGE_TYPE_NV21 

Raw YUV 4:2:0 image with a plane of 8 bit Y samples followed by an interleaved V/U plane containing 8 bit 2x2 subsampled chroma samples. Typically, frames from Android video capture are obtained in this format.
